📊 Intervals.icu Coaching Framework

This repository provides a structured coaching decision system for endurance sports. It combines data-driven metrics, subjective feedback, and heuristics into a repeatable framework for analyzing athlete readiness, load management, and long-term progression.

📂 Repository Structure

Coaching Heuristics Pack.md
Core decision rules, periodisation logic, recovery thresholds, and sport-specific adaptations.

Coaching Cheat Sheet.md
Quick-reference traffic-light thresholds for fast classification of training load, recovery, and training quality.

Advanced Marker Reference.md
Definitions and thresholds for derived training markers such as ACWR, Strain, Monotony, Polarisation Index, Durability Index, Recovery Index.

Glossary & Placeholders.md
All variable placeholders used in reports, including metrics, trends, actions, and events.

Coach profile.md
Skills summary and technical background of the coaching approach, including frameworks applied (Seiler 80/20, Banister TRIMP, Foster’s Monotony/Strain, tapering models).

🧠 Core Concepts

Load Management → CTL, ATL, ACWR, Monotony, Strain

Recovery → HRV, Resting HR, Sleep, Recovery Index

Training Quality → Polarisation Index, Durability Index, Quality Session Balance

Periodisation → Build, Overload, Deload, Consolidation

Event Prep → Taper protocols, race-specific sharpening

🚦 Workflow

Input Data → Training load, wellness, subjective feedback.

Markers → Derived metrics computed (ACWR, Monotony, Strain, Polarisation Index, etc.).

Heuristics → Apply decision rules from the Coaching Heuristics Pack.

Classification → Use the Coaching Cheat Sheet for fast traffic-light decisions.

Reports → Assemble athlete-facing reports with placeholders defined in the Glossary.

🏋️ Supported Sports

Triathlon (Ironman, Short-Course)

Cycling (Gran Fondo, Road, TT)

Running (Marathon, Ultramarathon, 10k)

Each sport profile includes volume norms, taper guidelines, and key session structures.
